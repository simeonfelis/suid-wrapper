suid-wrapper
============

C wrapper for scipts, as glibc dropps suid permissions in scripts (everything beginning with #!/bin...)

See main.c for licence.

Prerequesites:
  *  gnu make
  *  c compiler

Settings:
=========

See makefile. Check main.c for correct uid and gid.

Install:
========

  *  make install

Uninstall:
==========

  *  make uninstall
